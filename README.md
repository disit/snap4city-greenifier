# Greenifier Antwerp

App to promote public transport in Antwerp, with gamification features.

# Features

See [greenifier.pdf](https://antwerp.greenifiers.net/greenifier.pdf).

This app was written for the [Snap4city](http://snap4city.org/) Hackathon 2019.
